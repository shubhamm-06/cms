# Project Overview

CurateMyStay helps manage short-stay properties, bookings, expenses, owner payouts, and owner queries.

## Roles

Company Admin can manage users, properties, bookings, expenses, P&L, payouts, queries, settings, and their own profile.

Property Owner can view only their assigned property data: property details, bookings with guest names and phone numbers, property expenses, payouts, simple queries, and their own profile. Owners can approve their previous-month payout or raise a payout query during the review window.

## Phase 1 Scope

- Reference-matched public landing page based on `reference/index.html`
- Email/password and Google auth through Supabase
- Pending and inactive account screens
- Admin dashboard modules
- Owner dashboard modules with read-only operational records and payout review actions
- Supabase schema and RLS-focused security notes

## Later

Calendar, Forma Sheet, budgeting, channel sync, payment integration, receipt uploads, advanced disputes, scheduled payout jobs, and owner statement PDFs are intentionally left for later.
